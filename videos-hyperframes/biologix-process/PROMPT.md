# Prompt profesional — Video de proceso Biologix

Crear un video horizontal 1920x1080, sobrio y profesional, para Lumora Partner. El objetivo no es vender “IA”, sino evidenciar visualmente cómo una orden de compra se convierte en un proceso operativo conectado para Biologix Colombia SAS.

## Dirección visual
- Mantener la paleta de Lumora Partner: fondo navy oscuro (#0A0F1F, #0E1630), paneles #112044, acentos azules #1D4ED8, #3B82F6, #93C5FD y texto #EAF2FF.
- Estética: dashboard operacional premium, cards flotantes, conectores, estados de procesamiento, interfaz tipo sistema real.
- Evitar exceso de texto. La historia debe entenderse por elementos visuales: orden de compra, extracción de datos, validación, notificaciones, dashboard.
- Transiciones sobrias: velo oscuro/fade suave con una línea de luz, nada de wipes grandes ni movimientos exagerados.
- Ritmo lento y legible: cada escena dura mínimo 10 segundos.

## Estructura narrativa

### Escena 1 — Orden recibida (10s)
Mostrar una card grande simulando una orden de compra de Biologix: cliente, finca, productos, cantidad, fecha y estado “Recibida”. Al lado, una bandeja de entrada/correo con confirmación de recepción. El mensaje clave: “La orden entra al sistema”.

### Escena 2 — Procesamiento automático (11s)
Mostrar la orden conectándose con un núcleo Lumora mediante líneas animadas. El sistema extrae campos: cliente, finca, productos, responsable, canal de notificación. Usar chips y estados: “Extraído”, “Validado”, “Listo para notificar”. El mensaje clave: “Los datos se organizan sin copiar y pegar”.

### Escena 3 — Notificaciones y equipo en campo (11s)
Mostrar cards de WhatsApp/correo para cliente, equipo Biologix y jefe de finca. Las notificaciones salen desde el sistema hacia cada actor. Al fondo, mapa/ruta de finca. El mensaje clave: “Cada persona recibe la información correcta”.

### Escena 4 — Dashboard vivo y decisión comercial (10s)
Mostrar dashboard responsive con alertas RFM híbridas: “Cliente fuerte”, “Cambio de comportamiento”, “Visita recomendada”. Incluir un mockup móvil para uso en ruta. Cierre con Lumora Partner: “Procesos reales. Sistemas conectados.”

## Reglas de motion
- Entradas suaves, con stagger lento y respiración visual.
- No saturar la pantalla con texto simultáneo.
- Cada escena debe tener un momento estable de lectura de al menos 5 segundos.
- Las transiciones no deben robar protagonismo al contenido.
- Video sin audio por ahora.
